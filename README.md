# react-native-vpn2sock

vpn2socks package for react-native

## Installation

```sh
npm install react-native-vpn2sock
```

## Usage

```js
import Vpn2sock from "react-native-vpn2sock";

// ...

const result = await Vpn2sock.multiply(3, 7);
```
